# Abap-Demo
My first ABAP Demo Project 🚀
This repository contains my first ABAP project, created as part of my learning journey into SAP development and modern ABAP tooling.

# Description
The project features a fundamental ABAP class (ZCL_HELLO_GIT_DEMO94) designed to output a message to the ABAP Console. The primary goal was to practice:

Object-Oriented ABAP syntax.

Working within the Eclipse ADT (ABAP Development Tools) environment.

Integrating SAP development with GitHub using abapGit.

# Technologies & Tools
Language: ABAP

IDE: Eclipse ADT

Version Control: GitHub & abapGit

Platform: SAP BTP / ABAP Environment

# How to Use
Clone this repository into your SAP system using abapGit.

Activate the class 

Run the class as an ABAP Application (Console) by pressing F9 in Eclipse.
